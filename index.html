ABOUT CROPPER;



Initiate cropping by creating a new SelectImage() object. Example:
var crop=new SelectImage(para1,para2,para3); where para1
is the input type="file" element, para2 is a previewing div element and para3 is the height of the final output image e.g 200

Functions


ResetView(length); >> Used to reset the length of the image that will be output in your previewing div element. Its always a square and only the length is required as the parameter. This does not affect the length of the final image.

DebugCrop(); >> Used to monitor errors by the programmer.


Events


oncropped event It is on this event the the cropped image is ready and further actions can be done on image like uploading;
Example:
//Assign f the input element, "input type'file' "
//
var f=document.getElementById("file");

//Previewing div element
var preview=document.getElementById("preview");

//Set your view as follows
//
ResetView(300);

//Actual cropping happens here
//
var crop=new SelectImage(f,preview,200);

//Manipulate your cropped image here
//width and height are equal because it's a square
//
crop.oncropped=function (e){
var image=e.data;
var height=e.data.height;
var width=e.data.width;
}
